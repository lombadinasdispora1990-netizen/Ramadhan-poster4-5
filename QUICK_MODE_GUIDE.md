# 🎨 Quick Guide: New Generation Modes

## What's New?

BarokahGen sekarang memiliki **6 mode transformasi** yang bisa kamu pilih! Dari foto realistis, kartun 3D, anime, lukisan minyak, hingga cyberpunk!

---

## 🚀 Cara Menggunakan

### 1. Upload Foto
Upload fotomu seperti biasa (drag & drop atau browse)

### 2. Pilih Mode Transformasi
Setelah upload, akan muncul **6 kartu mode** di bawah tombol Generate:

```
┌─────────┬─────────┬─────────┬─────────┬──────────┬──────────┐
│ 📝      │ 🖼️      │ 🎭      │ 🌸      │ 🎨       │ ⚡       │
│ Text    │ Realistic│ 3D      │ Anime   │ Painting │ Cyberpunk│
│ Only    │         │ Cartoon │         │          │          │
└─────────┴─────────┴─────────┴─────────┴──────────┴──────────┘
```

**Klik salah satu kartu untuk memilih mode!**

### 3. Isi Detail
- Nama pengirim (opsional)
- Jenis ucapan (Formal, Lucu, Santai, dll)

### 4. Generate Poster
Klik tombol generate dan tunggu AI bekerja!

### 5. Download
Download poster unikmu dalam style yang dipilih!

---

## 🎨 Penjelasan Mode

### 📝 Text Only
- **Fungsi**: Hanya generate teks, foto tetap asli
- **Cepat**: ~2-5 detik
- **Kapan**: Kalau mau pakai foto asli tanpa perubahan

### 🖼️ Realistic (Default)
- **Fungsi**: Ubah ke busana Islami yang realistis
- **Pria**: Baju koko + kupiah
- **Wanita**: Gamis + jilbab
- **Waktu**: ~5-8 detik

### 🎭 3D Cartoon
- **Style**: Pixar/Disney seperti Toy Story, Frozen
- **Fitur**: Mata besar, proporsi chibi, warna cerah
- **Keren**: Cocok untuk poster ceria!
- **Waktu**: ~6-10 detik

### 🌸 Anime
- **Style**: Anime Jepang seperti Studio Ghibli
- **Fitur**: Mata detail, shading lembut, aesthetic
- **Keren**: Buat penggemar anime!
- **Waktu**: ~6-10 detik

### 🎨 Oil Painting
- **Style**: Lukisan minyak klasik Renaissance
- **Fitur**: Goresan kuas terlihat, warna hangat
- **Keren**: Terlihat seperti lukisan museum!
- **Waktu**: ~6-10 detik

### ⚡ Cyberpunk
- **Style**: Sci-fi futuristik tahun 2077
- **Fitur**: Lampu neon, pakaian tech-wear
- **Keren**: Unik dan edgy!
- **Waktu**: ~6-10 detik

## 💡 Tips Tiap Mode

### Text Only
✅ Pakai kalau puas dengan foto asli
✅ Paling cepat
✅ Hemat quota API

### Realistic
✅ Gunakan foto berkualitas tinggi
✅ Pencahayaan bagus
✅ Hadap depan

### 3D Cartoon
✅ Cocok untuk foto anak-anak
✅ Warna cerah lebih bagus
✅ Ekspresi senyum lebih baik

### Anime
✅ Angle 3/4 atau samping works well
✅ Ekspresi natural
✅ Lighting lembut

### Oil Painting
✅ Pose formal lebih cocok
✅ Background netral
✅ Potret klasik

### Cyberpunk
✅ Foto gelap bisa work
✅ Look modern/edgy
✅ Ekspresi bold

## ⚠️ Troubleshooting

**Mode tidak berubah?**
- Refresh halaman

**Transformasi lama?**
- Cek koneksi internet
- Coba foto lebih kecil (<2MB)

**Hasil tidak sesuai?**
- Coba mode lain
- Pastikan foto jelas

## 🎯 Contoh Penggunaan

### Skenario 1: Poster Tradisional
1. Upload foto
2. Pilih **Realistic**
3. Pilih gaya **Formal/Religius**
4. Generate → Download

### Skenario 2: Poster Lucu untuk Teman
1. Upload foto teman
2. Pilih **3D Cartoon**
3. Pilih gaya **Lucu/Ngakak**
4. Generate → Download

### Skenario 3: Poster Estetik
1. Upload foto portrait
2. Pilih **Oil Painting**
3. Pilih gaya **Formal/Religius**
4. Generate → Download

### Skenario 4: Poster Unik
1. Upload foto
2. Pilih **Cyberpunk**
3. Pilih gaya **Santai/Hangat**
4. Generate → Download

### Skenario 5: Tetap Pakai Foto Asli
1. Upload foto
2. Pilih **Text Only**
3. Pilih gaya apapun
4. Generate → Download
